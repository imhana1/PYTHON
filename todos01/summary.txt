1. 웹은 요청과 응답만 존재 - 상태를 기억하지 못한다

2. 사용자 정보 저장
    - cookie : 사용자 컴퓨터에 저장 - 중요하지 않은 간단한 정보(방문했던 링크)
    - session : 웹서버 메모리에 저장 - 로그인 정보 (서버를 껐다 키면 사라짐)
    - DB : 별도의 데이터베이스 서버에 저장 - (영구 저장)

3. session
    - 서버 메모리에 있는 사용자 정보 저장 영역 (사용자 마다 주어진다)
    - 사용하지 않고 일정 시간이 지나면 파괴
    - 요청을 보낸 사용자가 세션이 없을 때 만들어짐
    - 로그인 : session['username'] = 로그인 아이디
    - 로그인 확인 : session.get('username') != None
    - 로그아웃 : session.clear()
